# 项目简介

## 核心功能
- 用户传播力评估
- 热门话题追踪
- 评论转发点赞三维交互建模
- 真实用户KOL等级打分
- 数据图表展示

## 适用场景
- 社交网络舆情分析
- 用户行为研究
- 市场营销策略制定
- 网络营销效果评估

## 技术特点
- 全链路数据爬取，全量历史内容还原
- 利用 PySpark 实现高效清洗、聚合、统计
- 用户画像、影响力指数、互动趋势一站式分析
